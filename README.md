# 知识笔记

## Python篇

1.for循环是以一个代码块来结算的，每一次变量代入都循环一次

2.while循环是反向判定条件，直到不满足条件才结束循环

3.控制流的时序错位，做好初始变量表达式

4.if语句是满足条件即进入分支

5.代码初始化，写出变量。字符串，列表等

6.循环嵌套思维

7.break语句在while语句中可以用来限制边缘条件

8.变量查找替换可以通过代入变量赋值来实现（python特色），还有计数器

9.善用逻辑符号和布尔代数

10.函数中，重点是实参与形参

- 嵌套循环和递归，重点是return返回值和边界条件控制。需要用if来规定最简单的边界条件的返回值
